Что экшен должен сделать:
Переместить файл из одной директории в другую

Параметры:
sourcePath - Путь до файла, который необходимо переместить
destDirPath - Папка, куда будет перемещен файл
newFileName - Новое имя для перемещенного файла
overwrite - Перезаписать существующий файл. True для перезаписи файла, иначе False
